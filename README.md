# Delicious-Stores-App

Delicious-Stores-App is a React Application for Restaurants. Its purpose is to: 

 1. Let the user search for a store(use the map or browse the collection of stores). 
 2. Let an owner of a store to add the store to a collection of stores, make it visible and advertise it.
